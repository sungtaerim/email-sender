# Получение списка уволенных сотрудников, активных в Teamcenter, в формате .pdf и отправление его по email

По умолчанию отчёт отправляется в 10.00 1 и 16 дня каждого месяца. 
Получатели отчёта по умолчанию устанавливаются в application.properties

**При необходимости внеочередного получения отчёта следует выполнить запрос:**
    
> http://{server_name}:{port}/email-sender-1.0/email

{server_name} - имя сервера, на котором запущено приложение.
{port} - номер порта. 
Отчёт будет направлен пользователю по умолчанию. В отчёте будут включены сведения об уволенных сотрудниках, чьи учётные записи активны в Teamcenter.

**При необходимости получения отчёта другим пользователем следует выполнить запрос:**

> http://{server_name}:{port}/email-sender-1.0/email/{user_id}

{user_id} - ID пользователя, которому нужно получить отчёт. ID можно писать в нижнем регистре.

Для получения отчёта с полным списоком уволенных сотрудников следует добавить параметр status=1.
